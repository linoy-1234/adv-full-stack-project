Onco+Log is a comprehensive, production-ready Full-Stack medical web application built to empower cancer patients during active treatment while simulating a secure hospital information system integration. The platform acts as a protective digital environment that bridges the gap between raw clinical data and real-time patient management. By automating clinical data retrieval, the system eliminates manual entry errors for critical records like blood work, medications, and chemotherapy cycles. It provides automated predictive metrics, surfaces complex correlations between medications and daily side effects, and gives oncologists a dedicated, secure clinical portal to officially evaluate data and authorize upcoming treatment cycles.
